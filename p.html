Tentang Saya

Saya mahasiswa Politeknik Negeri Jember yang sedang aktif belajar di bidang pengembangan web backend menggunakan PHP dan MySQL. Masih dalam proses membangun pemahaman — bukan developer berpengalaman, tapi serius menekuni prosesnya.

Ketertarikan saya berkembang ke arah sistem manajemen jaringan, khususnya RT/RW Net. Dari sana saya mulai mempelajari FreeRADIUS dan bagaimana integrasinya dengan aplikasi web bisa membantu pengelolaan ISP skala kecil menjadi lebih terstruktur.

Saya juga memiliki ketertarikan pada dasar-dasar keamanan aplikasi — belajar bagaimana celah umum seperti SQL Injection dan XSS bekerja, agar bisa membangun sistem yang lebih aman dari sisi kode.

Catatan: Semua yang ada di sini merupakan hasil belajar mandiri dan proyek latihan. Tidak ada klaim keahlian yang berlebihan.

Project

Web App Manajemen Usaha RT/RW Net Berbasis FreeRADIUS

Learning Project

Aplikasi web untuk membantu pengelolaan RT/RW Net, dibuat sebagai proyek pembelajaran. Terhubung dengan database FreeRADIUS untuk mengelola user dan layanan internet.

Fitur yang Dikembangkan
  • Generate voucher internet otomatis
  • Pengelolaan PPP (Point-to-Point Protocol)
  • Manajemen keuangan (pemasukan & tagihan)
  • Monitoring user aktif secara real-time
Teknologi yang Digunakan
PHP MySQL FreeRADIUS Bootstrap 5 HTML CSS JavaScript
Status
  • Dalam pengembangan aktif
  • Belum production-ready
